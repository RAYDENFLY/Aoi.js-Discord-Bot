# Aoi.js-Discord-Bot
di buat menggunakan API Aoi.js

**How to install**
1. rename `example.env` to `.env`
2. write your tken bot in `.env`
3. change prefix in `index.js`

**How to run**
1. `npm i`
2. `npm start` / `node index.js`
